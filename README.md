### Cara menjalankan program
1. jalankan program dengan menulis di terminal :
```
python3 solver.py
```
2. pilih input puzzle (random atau dari file)
```
Choose your puzzle : 
3. Random puzzle
4. Puzzle from file
5. Exit
```
3. jika memilih input dari file, masukan nama file test yang tersedia
```
Enter the file name : 
puzzle5S.txt
```
